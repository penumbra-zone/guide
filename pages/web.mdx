# Using Penumbra from a web browser

This section of the guide describes in detail how to use Penumbra from a web browser.

<Callout type="info" emoji="ℹ️">Get up and running quickly with [our interactive guide](/quickstart) to basic Penumbra actions!</Callout>

For more detailed information on each of the actions, see the following pages:

* [Prax Wallet](./prax.mdx) describes how to install a browser extension that supports Penumbra.
* [IBC Transfers](./ibc-transfers.mdx) describes how to shield funds from other chains.
* [Balances](./balances.mdx) describes how to view your account's balances.
* [Sending Funds](./send.mdx) describes how to send funds to another address.
* [Swapping Funds](./swap.mdx) describes how to swap funds to exchange one asset for another.
* [Staking](./stake.mdx) describes how to delegate your UM staking token to a validator to receive staking rewards.
* [Governance](./vote.mdx) describes how to participate in governance.
* [Disconnecting Prax](./disconnect.mdx) describes how to disconnect Prax Wallet from frontends.
