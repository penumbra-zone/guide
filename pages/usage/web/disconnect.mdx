# Disconnect Prax Wallet

This section describes how to disconnect Prax Wallet from a particular frontend.

First, click on the Prax Wallet extension icon in the Chrome toolbar, and select the Settings menu, which is the hamburger menu icon â˜° in the upper left corner:

<div className="flex justify-center">
  ![Prax Wallet Settings](/images/prax-settings.png)
</div>

Then, select **Connected Sites**:

<div className="flex justify-center">
  ![Connected Sites](/images/prax-connected-sites.png)
</div>

Then, click the trash button next to the site URL you want to disconnect from.

That's it! You are now disconnected from that frontend.
